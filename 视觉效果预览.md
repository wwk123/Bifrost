```css
first_rank         → 金色系 (from-yellow-400 to-yellow-600)
achievement_unlock → 紫色系 (from-purple-500 to-pink-600)
overtake_friend    → 蓝色系 (from-blue-500 to-cyan-600)
milestone_reached  → 橙色系 (from-orange-500 to-red-600)
```

```css
10 BNC    → 灰色   #94a3b8  (普通)
```

## 🎯 用户体验细节

### 微交互
1. **按钮悬停** - scale(1.05) + shadow
2. **按钮点击** - scale(0.95)
3. **卡片进入** - 从右滑入 + 淡入
4. **卡片退出** - 向右滑出 + 淡出

### 视觉层次
1. **庆祝动画** - z-index: 50 (最高)
2. **社交动态** - z-index: 40
3. **转盘** - z-index: 1
4. **背景** - z-index: 0
