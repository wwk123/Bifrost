- 窗口大小：1920x1080
- 关闭所有无关标签页
- 开启声音（用于展示音效）

## 💡 演示技巧

### Do's ✅
1. **保持节奏** - 不要操作太快，给观众时间观察
2. **等待动画** - 让动画效果完全展示
3. **互动提问** - "大家觉得这个效果怎么样？"
4. **展示细节** - 鼠标悬停效果、过渡动画
5. **强调数据** - 用具体数字说话

### Don'ts ❌
1. **避免刷新页面** - 可能丢失状态
2. **不要跳过动画** - 动画是亮点
3. **避免同时开太多功能** - 容易混乱
4. **不要忽略移动端** - 可以展示响应式

## 🚨 应急预案

### 问题1：网络延迟导致加载慢
**解决：** 提前加载所有页面，保持标签页打开

### 问题2：动画卡顿
**解决：** 关闭其他应用，使用Chrome硬件加速

### 问题3：LocalStorage数据异常
**解决：** 准备清理脚本
```javascript
// 在Console执行
localStorage.clear();
location.reload();
```

### 问题4：功能未按预期工作
**解决：** 切换到备用演示视频（提前录制）
